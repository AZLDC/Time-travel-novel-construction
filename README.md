# 格物治世

架空古代 × 替嫁 × 甜寵日常 × 格物技術線的長篇小說專案。

本檔案是專案的入口說明，給作者與 AI 使用。

---

## AI / 作者入口

> 在新環境第一次開啟本專案時，請**先讀完本節**，再依指引打開對應設定檔。

1. **L1 極簡入口：`Settings/AI壓縮包.md`**  
   - 單一壓縮檔，包含：世界觀概要、主要人物簡表、發明進度、風格要點、創作原則  
   - 給所有 AI：**若 Context 有限，優先只讀這一份就可以開始寫作**

2. **L2 詳細索引：`Settings/AI索引.md`**  
   - 含：人物/伏筆/事件/發明等各種索引表、事件素材池、AI 寫作檢查清單、Settings 檔案對照  
   - 需要查 ID、查伏筆、查具體設定、或做完稿檢查時使用

3. **L3 詳細設定與故事大綱**  
   - `Settings/故事設定或大綱.md`：故事核心、人物關係、劇情大綱、寫作風格與制度設定  
   - `Settings/人物設定.md`：人物背景與性格細節  
   - `Settings/架空世界設定.md`：世界觀完整說明  
   - 其餘 `Settings/` 檔案依 `AI索引.md` 中的「Settings 檔案對照」查表使用

4. **章節正文與章節卡**  
   - 正文章節：`Story/` 下的 `第一章.md`、`第二章.md`……  
   - 章節卡：`章節卡/` 下的 `第N章.md`，每章含「規劃區＋摘要區」  
   - 寫第 N 章時：先讀 `章節卡/第N-1章.md`、`章節卡/第N章.md`，再動筆寫 `Story/第N章.md`

---

## 目錄速覽

```text
格物治世/
├── README.md              ← 本檔案（專案入口說明）
├── Story/                 ← 正文章節（第一章.md、第二章.md...）
├── Settings/              ← 世界觀、人物、事件、發明等設定檔
│   ├── AI壓縮包.md        ← L1：極簡入口（AI 首選）
│   ├── AI索引.md          ← L2：索引與檢查清單
│   ├── 故事設定或大綱.md  ← 劇情大綱與格局設定
│   ├── 人物設定.md        ← 人物背景與關係
│   ├── 架空世界設定.md    ← 世界觀完整說明
│   ├── 因果紀錄.md        ← 角色當前所在地與可用性
│   └── 事件備案.md        ← 單元劇模板與事件素材
├── 章節卡/                ← 各章規劃卡（第N章.md：規劃＋摘要）
└── 試驗圖/                ← 圖片素材（非寫作用）
```

> 提示：若未來在其他電腦或新環境啟動 AI，**請先打開 `README.md`，再依本檔指引讀取 `Settings/AI壓縮包.md` 等檔案**，即可在沒有雲端記憶的情況下恢復完整設定。
