## 四、劇情大綱

### 第一章：大婚

**皇帝早朝**
- 前線傳捷報，陳家鎮守邊關大捷
- 皇帝宣布陳、藍兩家聯姻，三個月內於京中陳國公宅邸完婚
- 口諭：陳子鑑完婚後不隨父兄出征（實為留京質子）
- 皇帝意圖：藍家勢大、陳家功高，令才女嫁紈褲以打壓兩家

**藍府密謀**
- 藍相一籌莫展，黃氏哭鬧後密謀替嫁
- 黃氏哄騙年幼的盈盈，瞞著所有人安排替嫁
- 嬤嬤教導盈盈婚禮流程：下轎、拜堂、洞房規矩
- 特別叮囑：進洞房前不做聲、不亂動（隱藏年幼特徵）

**婚禮當日**
- 藍府：嫁衣做兩套，藍憶雲著嫁衣見客，實際上轎者為盈盈
- 陳府：張燈結彩、賓客絡繹
- 迎親鬧酒（擋門酒、上馬酒、下馬酒）：子鑑被灌醉，酒中混入蘇家毒藥
- 蘇家少爺愛慕藍憶雲，欲毒殺陳子鑑破壞賜婚

**拜堂與穿越**
- 子鑑酒醉昏迷、毒發，被架著完成拜堂（全程無法說話）
- 送入洞房前原主毒發身亡，現代男主穿越入體
- 半昏半醒被送入洞房

**洞房夜**
- 盈盈偽裝：嫁衣墊肩藏身形、長裙遮雙腿、手套藏小手
- 盈盈緊張複習流程，男主接收原主記憶
- 交杯酒、遞竹棍、掀蓋頭
- 男主看清盈盈年幼模樣，驚得站起，盈盈嚇得發抖
- 男主心生憐憫，退至外房，承諾不圓房、成年後再議
- 當晚兩人分睡裡外兩床

### 第二章：新婚

**第二天早晨**
- 盈盈為男主更衣（按習俗）
- 男主偽裝落紅：酒水+紅紙染被褥
- 盈盈鞋底墊軟布，偽裝初夜後走路不便

**請安與離別**
- 兩人向父兄請安
- 父兄當日離京返回邊關，帶走大部分僕從
- 府中只剩16名藍府送來的年幼女僕（黃氏不懷好意）

**困境浮現**
- 聖旨只賜婚，無官職、無俸祿
- 府中存銀不多，坐吃山空
- 替嫁之事傳遍京城，流言蜚語四起

### 第三章：第一桶金

- 男主利用現代知識製作精鹽、白糖
- 與商家合作，賺取第一桶金
- 改善府中生活，寵溺盈盈

**契約制度**：

- 願意繼續留下者仍可維持原契，主角對奴僕待遇極好，工作環境優良

### 玻璃製作技術（漸進改良）

#### 基礎配方
- **石英砂**（二氧化矽）：約七成，主要原料
- **草木灰**（碳酸鉀）：約兩成，助熔劑，降低熔點
- **石灰**：約一成，穩定劑，防止玻璃溶於水
- 熔煉溫度：需達1500度以上，持續數個時辰

#### 常見問題與改良

| 問題 | 原因 | 改良方案 |
|------|------|----------|
| 玻璃泛綠 | 普通沙含氧化鐵雜質 | 換用白色石英砂（河灘細篩）、或以磁石吸除鐵砂 |
| 氣泡多 | 溫度不足/時間太短/攪拌不均 | 延長高溫時間、加入少量硝石（澄清劑）幫助排氣 |
| 雜質顆粒 | 原料不純 | 石英砂多次過篩、草木灰水洗沉澱取上清再蒸乾 |
| 表面不平 | 冷卻不均 | 緩慢降溫、平板模具 |

#### 進階材料
- **硝石**（硝酸鉀）：少量添加可作為澄清劑，幫助氣泡逸出
- **白石英**：河灘或山中尋找純白色石英，含鐵量極低
- **提純草木灰**：將草木灰泡水過濾，取上清液蒸乾，可得較純的碳酸鉀

#### 改良順序（劇情用）
1. **首爐**：普通配方 → 泛綠、氣泡多、半透明
2. **二爐**：換白石英砂 → 顏色變淡，仍有氣泡
3. **三爐**：加硝石+延長時間 → 基本透明，氣泡減少
4. **四爐以後**：逐步優化，達到可製鏡品質

## 十一、寫作提示

### 創作目標
- **章節字數**：每章儘量超過萬字（10,000+）
- **總章數目標**：3,000 章

### 架空世界描述
  不要幾個章節就寫完世界設定，通過各章節內一些"不經意的提起"穿插，把架空世界設定的各方面一點點導入
  可以透過一些小事件刻意帶出世界相關的資訊
  
### 環境變化原則（重要）
- **不靠遷徙移動**：不以「搬到某城」「前往某國」推動劇情
- **靠社會影響變動**：
  - 主角推動某項工具/發明 → 社會產生連鎖反應
  - 主角向官方進言 → 政策變動、民間變化
  - 格物產品普及 → 百姓生活、階級觀念逐漸改變
- **核心邏輯**：環境變化是劇情推動的結果，而非角色物理位移
- 此為最大難度挑戰，需細膩鋪陳因果

### 大事件類型
- **天災**：寒災、旱災、蝗災、水災、瘟疫
- **人禍**：戰爭、貪官汙吏、世家作亂
- **社會變動**：難民遷徙、物價波動、民變等

### 章節管理原則
- **章節切割**：劇情按章節獨立切割，減少AI需讀取的資訊量
- **因果變更紀錄**：
  - 獨立維護「因果紀錄檔」（如 `Settings/因果紀錄.md`）
  - 記錄格式：`[章節] → [事件/伏筆] → [可能觸發章節/影響]`
  - 每章完成後更新，標註哪些伏筆已觸發、哪些待觸發
- **連鎖寫作技巧**：
  - 前期埋伏筆 → 後期章節觸發 → 產生連鎖反應
  - 伏筆可跨數十甚至數百章，需靠紀錄檔追蹤
